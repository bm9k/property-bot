## Installation
```
git clone https://github.com/bm9k/property-bot/
cd property-bot
npm install -g
```

## Usage
```
propertybot "42 Wallaby Way, Sydney 2000" -o results/
```
Note: the output directory will be relative to the current working directory, not the repository root
